Instalar la llibreria

    1. Anar a Sketch

    2. Import Library

    3. Manage libraries...

    4. Buscar "kinect"

    5. Instalar la: "SimpleOpenNI"

  • Clicar Install

  • Reiniciar Processing

Un cop instalada la llibreria faltarà el component per Windows de la Kinect

  • Descarregar-ho

  • Instalar-ho
